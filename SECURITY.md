# Security Policy

## Supported Versions

Only the latest version of this DWM fork is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a potential security vulnerability, please do not open a public issue. Instead, please report it via GitHub's private vulnerability reporting feature if enabled, or by opening a draft pull request with the fix.

### Hardened Security Model
This project follows a **High Entropy Security** model. If you find a way to bypass the isolated secret system or the strict file permissions enforced by the installer, please let us know immediately.

For more details on the built-in security features, see the [Security Model Guide](./docs/SECURITY.md).
